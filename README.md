# psikotes-bagus-dhabit-arziki
Website ini adalah aplikasi kuis interaktif mandiri (self-contained quiz web app) yang dirancang khusus untuk simulasi Psikotes Seleksi Masuk SMA. Website ini menggabungkan kesederhanaan penggunaan dengan fungsionalitas penuh tanpa memerlukan koneksi internet ataupun database eksternal.
